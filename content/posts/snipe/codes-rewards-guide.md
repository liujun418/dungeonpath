## How to Redeem Codes

1. Launch SNIPE from the Roblox app or website
2. From the main menu lobby, look for the **Settings gear icon** (bottom-right corner)
3. Click it to open the Settings panel
4. On the left sidebar of Settings, find and click the **Codes** button (it looks like a Twitter bird icon, or may be labeled "Codes")
5. In the text box, type or paste a code **exactly as shown** — capital letters matter
6. Click the green **Redeem** button
7. A popup confirms if the code worked. Rewards appear instantly

If a code fails:
- "Invalid Code" → Typo or case mismatch. Double-check spelling.
- "Expired Code" → The code is no longer active.
- "Already Redeemed" → You've used this code before on this account.

## What You Get From Codes

SNIPE codes grant four types of rewards:

### Credits (In-Game Currency)
Credits are the free currency used to buy weapon skins, charms, and cosmetic items from the in-game shop. Most codes give 100-500 Credits. A typical weapon skin costs 200-800 Credits, so 2-3 codes can get you a free skin.

### Weapon Skins & Wraps
Some codes give exclusive skins that can't be purchased with Credits. These are usually event-themed (seasonal, milestone celebrations) and become rare once the code expires. The Spring Camo Skin from `SPRINGUPDATE` is a current example.

### Ability Unlocks
The `WALLRUN` code permanently unlocks the Wall Run ability without spending Credits. This is the most valuable type of code since abilities normally cost 1,500-3,000 Credits each.

### XP Boosts
Temporary double XP multipliers (usually 1-2 hours). Best activated right before a long play session. XP boosts apply to both account level and battle pass progress.

## Tips for Code Hunters

1. **Codes expire fast.** Event codes last 48-72 hours. Milestone codes last 2-4 weeks. Don't save codes for later — redeem immediately.
2. **One use per account.** Every code can only be redeemed once per Roblox account. No exceptions.
3. **Case matters.** `snipe100m` won't work. `SNIPE100M` will. Copy-paste is your friend.
4. **No third-party code generators.** Any website or video claiming to generate unlimited SNIPE codes is a scam trying to steal your Roblox account. Codes only come from the developer.
