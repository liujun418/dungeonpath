> *Last updated: May 10, 2026. This guide covers every resource type in Palworld, where to find it, how to farm it efficiently, and which Pals are best for gathering each material.* See also: [Base Building Guide](/posts/palworld/base-building-guide/), [Pal Locations Guide](/posts/palworld/pal-locations-guide/).

## Resource Categories

Resources in Palworld fall into several categories:

| Category | Examples | Primary Use |
|----------|----------|-------------|
| Basic Building | Wood, Stone, Fiber | Structures, tools, early crafting |
| Ores and Metals | Ore, Coal, Refined Ingot, Pal Metal | Tools, weapons, armor, crafting stations |
| Spheres and Capture | Paldium Fragment, Pal Sphere | Capturing Pals (entire sphere line) |
| Fuel and Energy | Wood, Coal, High Quality Pal Oil | Powering furnaces, generators |
| Food and Farming | Berries, Wheat, Milk, Egg, Honey | Feeding Pals, crafting cakes, cooking |
| Mechanical | Polymer, Carbon Fiber, Circuit Board | Endgame weapons, armor, technology |
| Organic | Pal Fluids, Leather, Organ, Horn | Crafting, spheres, medicine |
| Ancient | Ancient Civilization Parts | Ancient Technology, high-tier items |
| Ranch | Wool, Milk, Egg, Honey, Pal Fluids | Passive production from Pals |
| Special | Lifmunk Effigy, Ruby, Technical Manual | Permanent upgrades, selling |

## Basic Building Resources

### Wood

| Detail | Information |
|--------|-------------|
| Found by | Chopping trees (anywhere), Ranch (some Pals), Lumbering Pals at base |
| Best farming method | Assign a Lumbering Pal (Warsect, Mossanda, Wumpo, Gorirat) to your base |
| Best farming location | Any forested area -- the base itself is the best farm |
| Uses | Structures, tools, campfire, charcoal production |
| Weight | 0.5 per unit |
| Stack size | 999 |

**Optimal Wood farming:** Assign a Pal with Lumbering level 3+ (Warsect, Wumpo, or Cryolinx) to a base in a forested area. A single lumbering Pal produces 500+ wood per hour passively.

### Stone

| Detail | Information |
|--------|-------------|
| Found by | Mining rocks (anywhere), Mining Pals at base |
| Best farming method | Assign a Mining Pal (Cattiva, Dumud, Digtoise) to your base |
| Best farming location | Any base location with stone nodes |
| Uses | Foundations, walls, Pal Spheres, stone tools |
| Weight | 1.0 per unit |
| Stack size | 999 |

### Fiber

| Detail | Information |
|--------|-------------|
| Found by | Harvesting bushes (hand), Gathering Pals, Lamball wool in Ranch |
| Best farming method | Ranch with Lamball (produces Wool, convert to Fiber via Cloth) |
| Best farming location | Base ranch |
| Uses | Cloth, Pal Spheres, saddles, armor |
| Weight | 0.1 per unit |
| Stack size | 999 |

### Paldium Fragment

| Detail | Information |
|--------|-------------|
| Found by | Mining blue-glowing rocks, ground pickups |
| Best farming method | Mine blue rocks around the starting area; they respawn |
| Best farming location | Grasslands (around starting beach), cave walls |
| Uses | Pal Spheres (ALL tiers), Palbox, Egg Incubator |
| Weight | 0.2 per unit |
| Stack size | 999 |

**Paldium farming route:** Starting beach -> head west along the cliffs. Approximately 20+ blue rock deposits in a 5-minute loop. Each deposit gives 5-15 Paldium. Repeat every 15 minutes as they respawn.

## Ores and Metals

### Ore (Regular)

| Detail | Information |
|--------|-------------|
| Found by | Mining rocky ore deposits (gray-brown with golden veins) |
| Best farming method | Base near ore cluster with Digtoise/Astegon |
| Best farming location | Plateau south of Rayne Tower (5+ nodes), Grassy Behemoth Hills |
| Smelts into | 1 Ore -> 1 Ingot (via Furnace + Kindling Pal) |
| Uses | Ingots -> tools, weapons, spheres, structures |
| Weight | 2.0 per unit |
| Stack size | 999 |

### Coal

| Detail | Information |
|--------|-------------|
| Found by | Mining dark-colored rock deposits in desert/volcano/snow biomes |
| Best farming method | Build a base near coal nodes with Mining Pal |
| Best farming location | Icy Weald, Desert cliffs, Volcanic foothills |
| Uses | Refined Ingot (Coal + Ore), Carbon Fiber |
| Weight | 3.0 per unit |
| Stack size | 999 |

**Coal + Ore base location:** The Icy Weald in the northeast has overlapping coal and ore deposits. Build your second or third base here for Refined Ingot production.

### Ingot

| Detail | Information |
|--------|-------------|
| Crafted at | Furnace (level 9) -- 2 Ore -> 1 Ingot |
| With Kindling | Jormuntide Ignis (Kindling 4) produces ingots fastest |
| Uses | Tools, weapons, structures, spheres, nearly everything |
| Weight | 3.0 per unit |
| Stack size | 999 |

**Production efficiency:** Jormuntide Ignis (Kindling 4) smelts 2x faster than Foxsparks (Kindling 1). Prioritize getting a high-level Kindling Pal.

### Refined Ingot

| Detail | Information |
|--------|-------------|
| Crafted at | Refined Furnace (level 18) -- 2 Ore + 2 Coal -> 1 Refined Ingot |
| Unlock | Technology level 14 (Refined Furnace) |
| Uses | High-tier weapons, armor, tools, structures |
| Weight | 5.0 per unit |
| Stack size | 999 |

**Production chain:** 
1. Mine Ore (ore nodes)
2. Mine Coal (coal deposits) 
3. Smelt in Refined Furnace with Kindling Pal
4. Each cycle: 2 Ore + 2 Coal = 1 Refined Ingot

**Tip:** Build your Refined Furnace base near both coal and ore deposits (Icy Weald or Desert Plateau edges).

### Pal Metal Ingot

| Detail | Information |
|--------|-------------|

| Unlock | Technology level 35 |
| Uses | Best endgame weapons and armor |
| Weight | 6.0 per unit |
| Stack size | 999 |

## Resource Farming Progression Roadmap

### Level 1-10: Manual Gathering
- **Wood/Stone:** Hand-punch or stone tools
- **Fiber:** Bush harvesting
- **Paldium:** Ground pickups + blue rocks
- **Berries:** Bushes
- **Pals for base:** Cattiva (transport/mining/handiwork)

### Level 10-20: Basic Automation
- **Wood:** First Lumbering Pal (Gorirat)
- **Ore:** Base near ore with Cattiva/Dumud
- **Food:** Berry Plantation + Watering Pal (Pengullet)
- **Ranch:** Chikipi (eggs), Lamball (wool)
- **Pals for base:** Digtoise (mining), Penking (watering)

### Level 20-35: Industrial Scale
- **Refined Ingots:** Coal + Ore base with Kindling Pal
- **Cake production:** Ranch (Chikipi, Mozzarina, Beegarde) + Wheat Farm
- **Sulfur:** Volcano runs with heat armor
- **Spheres:** Sphere Factory with Handiwork Pal
- **Pals for base:** Anubis (handiwork/bred), Jormuntide (watering), Jormuntide Ignis (kindling)

### Level 35-50: Endgame Production
- **Pal Metal:** Electric Furnace with Paldium + Refined Ingots
- **Polymer:** High Quality Pal Oil farming (Flambelle or ranch)
- **Carbon Fiber:** Coal in Electric Furnace
- **Circuit Board:** Pure Quartz + Polymer
- **Ancient Parts:** Alpha boss rotation farming
- **Pals for base:** Astegon (mining 4), Frostallion (cooling 4), Lyleen (medicine/planting)

## Farming Routes for Rare Resources

### Daily Boss Rotation (Ancient Parts)

1. Teleport to Desert Plateau -> defeat Paladius + Necromus (24-32 Ancient Parts) -- see the [Boss & Tower Guide](../palworld/boss-and-tower-guide.md) for detailed boss strategies
2. Teleport to Frostbound Mountains -> defeat Frostallion (8-12 Ancient Parts)
3. Teleport to Volcanic Crater -> defeat Jetragon (8-12 Ancient Parts)
4. Fly to Desert Tomb -> defeat Alpha Anubis (8-12 Ancient Parts)
5. Fly to Desert Oasis -> defeat Alpha Jormuntide (6-10 Ancient Parts)

**Total per hour:** ~60-80 Ancient Civilization Parts

### Sulfur Run (Volcano)

1. Enter volcano from east entrance
2. Follow the cave path counterclockwise
3. Mine all yellow deposits (20-30 nodes)
4. Exit and reset (deposits respawn in ~15 minutes)
5. **Yield per run:** 100-200 Sulfur

### Quartz Run (Snow Biome)

1. Teleport to Icy Weald fast travel
2. Head northeast along the mountain ridge
3. Mine all white crystal deposits (15-25 nodes)
4. Drop down and loop back
5. **Yield per run:** 80-150 Quartz

### Ore Loop (Grassy Behemoth Hills)

Best for early to mid-game, this location has 5+ ore nodes in close proximity. Build your second base here and assign a Digtoise or Astegon. It produces 200+ ore per hour passively.
