## Battle Pass System Overview

Blade Ball runs a seasonal battle pass system with new passes launching roughly every 30-40 days. Each pass features a free track and a premium track with exclusive abilities, sword skins, explosion effects, and limited cosmetics.

### How It Works

| Feature | Detail |
|---------|--------|
| Premium Pass Cost | 480-639 Robux (varies by season) |
| Season Length | ~30-40 days |
| Progress Reset | Full reset each season (nothing carries over) |
| Infinite Tiers (2026+) | Starting February 2026, passes have no level cap |
| Top Player Reward | #1 each season earns an exclusive item |
| Free Track | Basic currency and cosmetics |
| Premium Track | All abilities, exclusive items, bonus currency |

## Battle Pass Strategy

### Maximizing Season Progress

1. **Buy premium early** — the extra weekly challenges and +10 free levels compound over the full season
2. **Prioritize weekly challenges** — they give the most season XP. Complete all 3-4 each week
3. **Watch for 2x Battlepass Luck windows** — usually 48-hour events. Grind hardest during these
4. **Daily login** — even just logging in gives passive progress
5. **Infinite tier grind** (2026+) — top spot requires daily play. Set a realistic target tier and pace yourself

### Ability Selection Priority

When a pass has multiple abilities, prioritize unlocking them in this order:
1. Mobility abilities (movement, dash) — most universally useful
2. Offensive abilities (damage, AoE) — win more rounds
3. Defensive/utility abilities — situational but valuable
