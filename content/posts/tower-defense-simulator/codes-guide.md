## How to Redeem Codes

1. **Open Tower Defense Simulator** in Roblox
2. Look for the **Twitter Bird icon** on the right sidebar
3. Click it to open the code entry window
4. **Type the code** exactly as shown (codes are case-sensitive)
5. Press **Enter** or click **Redeem**

### Troubleshooting

| Error | Cause | Fix |
|-------|-------|-----|
| "Invalid code" | Typo or expired code | Copy the code exactly |
| "Already redeemed" | You used this code before | Each code works once per account |
| No reward appears | Code may be inactive | Check for new codes |

## Frequently Asked Questions

### Why are there no active codes right now?

TDS doesn't release codes on a fixed schedule. Codes are typically released during game updates, milestones, holidays, and special events. There can be weeks or months between code releases.

### Can I use codes multiple times?

No. Each code can only be redeemed once per account.

### Do codes work on all platforms?

Yes, codes work on all platforms where Roblox is available.

### What's the best code reward?

**Premium Crates** are the most valuable because they can contain rare towers or tower skins.
