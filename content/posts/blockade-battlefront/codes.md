## How to Redeem Codes

Blockade Battlefront uses two different redemption methods depending on the code type.

### Method 1: Chat Box (!redeem Codes)

For Cen codes using the `!redeem` format:

1. Join any Blockade Battlefront server
2. Click the chat box (top-left of screen, or press `/` to open)
3. Type the code exactly: `!redeem Return`
4. Press Enter
5. A confirmation message appears: "Successfully redeemed! +2,500 Cen"

### Method 2: Settings Menu (Skin/Gem Codes)

For non-`!redeem` codes like `100MVisit2` and `CamArmada`:

1. Click the **Settings** icon (gear/cog) on the side of the screen
   - On some versions, this is the **bird/Twitter icon**
2. Find the code redemption text box
3. Enter the code exactly (without `!redeem` prefix)
4. Click the **Redeem** button
5. Check your inventory or Gem balance to confirm
